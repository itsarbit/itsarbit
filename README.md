# Hi, I'm Arbit

I build infrastructure for AI agents - simulation, evaluation, and orchestration systems.

Currently building [Arklex](https://github.com/arklexai), where I work on:

- **[ArkSim](https://github.com/arklexai/arksim)** - Open-source agent simulation and evaluation framework. Generates realistic multi-turn conversations, scores agents on built-in and custom metrics, and produces visual reports. Protocol-agnostic (Chat Completions API, A2A).
- **[Agent-First Organization](https://github.com/arklexai/Agent-First-Organization)** - Agent framework for building production AI systems with an agent-first architecture.
